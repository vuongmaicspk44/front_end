CSS Units

- CSS units là những đơn vị được sử trong css để tùy chỉnh kích thước của những đối tượng trong website

- Những CSS Units hay được sử dụng px, %, rem, em, vw, vh

1. Absolute units - những đơn vị tuyệt đối

  • px

    • với màn hình độ phân giải thấp 1px tương đương với 1 điểm ảnh trên màn hình
    màn hình độ phân giải càng cao thì 1px càng tương đương với nhiều điểm ảnh

  • pt
  • cm
  • mm
  • inch
  • pc

2. Relative units- những đơn vị tương đối

  • %
  • rem

    • - rem sẽ phụ thuộc vào thẻ html

  • em

    • - em sẽ phụ thuộc vào thẻ gần nhất chứa nó mà chứa thuộc tính font-size
    - Trong thực tế ta sẽ sử dụng rem nhiều hơn vì nó dễ kiểm soát khi chỉ phục thuộc vào thẻ html

    - Thực tế ta sẽ để css thẻ html font-size: 100% tương đương với 16px(mặc định trình duyệt) và sử dụng css unit rem

  • vw

    • viewport width

  • vh

    • viewport height

  • vmin
  • vmax
  • ex
  • ch
